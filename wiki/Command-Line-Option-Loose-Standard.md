Loose Standards for Command Line Options
===


| option | description |
|--------|-------------|
| -a     | all         |
| -b     | buffer / block size |
| -c     | comand      |
| -d     | debug or delete |
| -D     | define |
| -e     | execute |
| -f     | file (input) |
| -h     | help |
| -i     | interactive / initialize |
| -I     | include |
| -k     | keep / kill |
| -l     | list / load / login / length / lock |
| -m     | message / mode |
| -n     | number |
| -o     | output |
| -p     | port / protocol |
| -q     | quiet |
| -r / -R | recurse |
| -s     | silent / subject |
| -t     | tag |
| -u     | user |
| -v     | verbose / version |
| -V     | version |
| -w     | width / warning |
| -x     | debug / extract |
| -y     | yes |
| -z     | compress |
